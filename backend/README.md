# Sync To-Do Backend

FastAPI implementation of the Sync To-Do REST API.

## Features
* Endpoints implemented exactly as per `docs/api-spec.json`:
  * `GET /tasks` — list all tasks.
  * `POST /tasks` — create new task.
  * `PATCH /tasks/{id}` — update existing task.
* In-memory storage by default (no external dependencies).
* Optional Cassandra layer can be enabled by setting the environment variable `USE_CASSANDRA=true` **and** having the `cassandra-driver` installed and a reachable Cassandra cluster.

## Requirements
* Python 3.11+
* (Optional) Cassandra 4.x if using persistent storage.

Install dependencies:
```bash
python -m venv .venv && source .venv/bin/activate
pip install -r backend/requirements.txt
```

## Running the server
```bash
# In-memory (default)
uvicorn backend.main:app --host 0.0.0.0 --port 8000

# With automatic reload during development
uvicorn backend.main:app --reload
```

To attempt Cassandra storage (stub—read-only demo):
```bash
export USE_CASSANDRA=true
uvicorn backend.main:app --reload
```

The API will be available at `http://localhost:8000`. Open `/docs` for interactive Swagger UI generated by FastAPI.

## Project Structure
* `backend/main.py` — application entrypoint.
* `backend/requirements.txt` — Python dependencies.
